# tp1

El archivo .gitignore es un archivo de texto plano que se ubica en la raiz del proyecto y tiene como funcion especificar que carpetas/archivos deben ser ignorados.

Es conveniente usarlo para evitar mandar al repositorio informacion confidencial como variables de entorno, Tokens, o tambien para mantener el repositorio mas limpio y no enviar archivos del SO o archivos temporales.
